# Custom Legcord Theme
## How to use
1. Download Legcord from here -> [https://legcord.app/](https://legcord.app/)  
2. Download image and css  
3. move to ``%appdata%/legcord/``, make folder named ``"userAssets"``

## Setting
### color  
48,92,222  

### background-image
